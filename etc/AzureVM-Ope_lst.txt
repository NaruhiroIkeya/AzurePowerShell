# Name          : AzureVM-Ope_lst.txt
# DESCRIPTION   : Describe machine information of the Azure VM server.
# USAGE         : <Azure VM Server name> <ResourceGroup> <Directory ID> <Backup Container>
# PARAMETERS    : <Azure VM Server name> :
#                     VM Server name recognized on Azure console.
#                 <ResourceGroup>        :
#                     Resource Group name registering VM Server.
#                 <Directory ID>         :
#                     The ID of the directory to which the "subscription that 
#                     created the ResourceGroup" belongs.
#                 <Backup Container>     :
#                     Container name registered for server backup.
#
# NOTE          : Described in Japanese below. If a file containing double-byte
#                 characters causes a failure, delete all Japanese lines.
#                 以下、日本語で記述。2バイト文字を含むファイルが障害の原因に
#                 なる場合、日本語の行をすべて削除すること。
# 
# <Azure VM Server name>
#    Azure 上に作成した仮想マシンのマシン名。Azureコンソール上で表示される名称か
#    Get-AzureRmVM コマンドで表示されるName欄のマシン名を記述する。
#    （大文字、小文字は気にしなくてもよい）
#    →各種処理の処理対象となるサーバを指定するのに使用する。
# <ResourceGroup>
#    対象の仮想マシンが所属するリソースグループ名。Azureコンソール上で対象の
#    仮想マシンの「概要」で表示される「リソースグループ名」か、Get-AzureRmVM
#    コマンドで表示されるResourceGroupName欄のリソースグループ名を記述する。
#    →各種処理の処理対象となるサーバを指定するのに使用するが、<Azure VM
#      Server name>だけでは対象を絞り切れない（異なるリソースグループで同名の
#      仮想サーバの作成が可能である）ため、作成先のリソースグループ情報が
#      必要となる。
# <Directory ID>
#    「リソースグループを作成した対象のサブスクリプション」を登録している
#    ディレクトリが持つ、ディレクトリID。
#    →各種処理を行う前の、Azureへのログインに使用。ログイン時にディレクトリID
#      を使用するため、「『対象サーバを登録しているリソースグループ』を作成した
#      サブスクリプション」が登録しているディレクトリのIDが必要となる。
#      ディレクトリIDはpowershellのコマンドでは確認できないため、Azureの
#      コンソール上で「Azure Active Directory」-「プロパティ」と辿り、表示される
#      「ディレクトリID」の値を確認すること。
# <Backup Container>
#    対象の仮想マシンのバックアップ実行用に設定した、バックアップコンテナーの
#    名称。（バックアップ先を指定するという意味合いで良いかも）
#    →バックアップ関連の各コマンドの実行時に必須。
#      スクリプト実行時のオプションがbackupの時だけ使用する。
# その他
#     各項目との区切り文字は、半角スペースを使用すること
# コメント行
#     # （半角のシャープ）が行の１文字目に記述されている場合はコメント行となり、
#     スクリプトがチェックする対象から除外される。
#---*----*----*----*----*----*----*----*----*----*----*----*----*----*----*----*
